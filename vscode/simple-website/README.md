# SimpleSite

A simple static website example (HTML, CSS, JS).

## Files
- `index.html` — main page
- `styles.css` — basic responsive styles with a dark theme
- `script.js` — small interactivity (theme toggle, greeting, contact form)

## Preview locally
Option 1 — Open in browser
- Open `index.html` directly in your browser.

Option 2 — Serve locally (recommended)
- Run a quick server: `python3 -m http.server 8000` from inside the `simple-website` folder
- Open http://localhost:8000 in your browser

Option 3 — VS Code Live Server
- Install the Live Server extension and click "Go Live".
